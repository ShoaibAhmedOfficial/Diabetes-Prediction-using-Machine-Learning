# Diabetes-Prediction-using-Machine-Learning

## Project Overview

This project uses Machine Learning to predict whether a person is diabetic or not based on medical attributes such as glucose level, blood pressure, BMI, and age. The model is trained on a diabetes dataset and predicts the likelihood of diabetes using classification techniques.

## Problem Statement

Diabetes is a common health problem worldwide. Early prediction can help in proper treatment and prevention.
The goal of this project is to build a Machine Learning model that can predict diabetes based on medical data.

## Technologies Used

## Machine Learning Workflow

The following steps were performed in this project:

1. Data Collection
2. Data Exploration
3. Data Visualization
4. Feature Selection
5. Data Splitting (Train/Test)
6. Feature Scaling
7. Model Training
8. Model Evaluation
9. Prediction System

## Model Used

Logistic Regression & SVC was used as the classification algorithm to train the model.

## Model Evaluation

The model was evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report

## Example Prediction

The model can take medical input values and predict whether a person is diabetic or not.

Example input:

Age, Glucose, BMI, Blood Pressure etc.

Output:

* Diabetic
* Not Diabetic
